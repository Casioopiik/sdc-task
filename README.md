# sdc-task

Úkol sloužící k prezentaci pro firmu Sykora Data Center

Spustitelné .jar je ve složce export do které se taktéž vkládá datový soubor. Pří spuštění aplikace je třeba jako parametr zadat název datového souboru. Z tohoto souboru je po spuštění aplikace vytvořen soubor export.txt, který obsahuje všechna prvočísla z datového souboru.

Projekt využíva pro kompilaci a spouštění Javu 8.
